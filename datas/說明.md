# 資料說明

- **001**：台電每月平均發電量（2016/01 ～ 2025/12）
    - 資料來源：https://data.gov.tw/dataset/29938
- **C-B0026-002**：每月平均資料（2016/01 ～ 2024/12）
    - 資料來源：https://opendata.cwa.gov.tw/dataset/climate/C-B0026-002
- **C-B0026-2025**：每月平均資料（2025/01 ～ 2025/12）
    - 2025 年資料從中央氣象局月資料頁面擷取：
        https://www.cwa.gov.tw/V8/C/C/Statistics/monthlydata.html
    - 格式同**C-B0026-002**

## 2. C-B0026-002 & C-B0025-2025 : 每月氣象

**檔案名稱**：`mn_Report_yyyymm.xml`，對應到yyyy年mm月的氣象數據

### 氣象資料結構表

| 主欄位 | 子欄位 | 說明 | 單位 |
|-------|--------|------|------|
| **station** | StationID | 氣象站編號 | - |
| | StationName | 氣象站名稱 | - |
| | StationNameEN | 氣象站英文名稱 | - |
| | StationAttribute | 氣象站分類 | - |
| **AirTemperature** | Mean | 該月份平均溫度 | ℃ |
| | Maximum | 該月最高溫 | ℃ |
| | MaximumDate | 該月最高溫發生日期 | - |
| | Minimum | 該月最低溫 | ℃ |
| | MinimumDate | 該月最低溫發生日期 | - |
| **Precipitation** | Accumulation | 該月累積降雨量 | mm |
| | GE01Days | 該月降雨量≥0.1mm天數 | day |
| **WindSpeed** | Maximum | 該月最大十分鐘風速 | m/second |
| | MaximumDate | 該月最大十分鐘風速發生日期 | - |
| **WindDirection** | Maximum | 該月最大十分鐘風向 | 360° |
| **PeakGustSpeed** | Maximum | 該月最大瞬間風速 | m/second |

## 3. combined_data.csv : 合併資料

將對應測站的每月氣象資料轉換並合併到發電資料中。
- 合併資料共 **3818 筆**（包括所有發電站、所有月份資料）
- 欄位 1-6 來自 **001.csv**
- 欄位 7 來自 **plant_station_mapping.csv**（紀錄每個電廠對應的最近測站）
- 欄位 8-25 來自氣象 **XML 檔案**（C-B0026-002 及 C-B0026-2025）
